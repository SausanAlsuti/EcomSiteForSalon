About Lamsaty Saloon

This project demonstrates how to create a simple website for a Lamsaty Saloon where users can book services and contact the salon directly via email. Using basic technologies like HTML, CSS, JavaScript, and PHP, we show how to handle service bookings and contact messages without using any external libraries or frameworks.

The project includes a basic service booking form where users can choose a service, enter their details, and submit their booking. The data is then sent via PHP's mail() function to the salon’s email address, ensuring that bookings are received in real-time.

In addition to service booking, there’s a simple contact form that allows users to send messages directly to the salon. This also uses the mail() function to send the user’s message via email.

This project is perfect for beginners who want to learn how to build a small, functional website from scratch with basic web technologies.
